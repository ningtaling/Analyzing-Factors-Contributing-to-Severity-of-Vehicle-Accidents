# Vehicle Accident Severity Analysis

## Project Overview
This project analyzes factors contributing to the severity of vehicle accidents using real-world crash data from Montgomery County.

## Key Questions
- How does speed limit affect injury severity?
- Does vehicle age increase accident risk?
- How does driver fault impact outcomes?

## Methods
- Exploratory Data Analysis (EDA)
- Logistic Regression (GLM)
- LDA & QDA comparison
- K-fold Cross Validation (k=50)
- Bootstrap validation

## Key Findings
- Higher speed limits are associated with more severe injuries
- Older vehicles are more likely to be involved in serious/fatal accidents
- Driver fault significantly increases severity risk

## Tools
- R
- ggplot2
- Statistical modeling
